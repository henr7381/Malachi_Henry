This is a project for ECE--363-01-SP-2023 - Logic & Computer Design.
This is a 32 bit MIPS CPU, with a custom instruction set. 

You can find a download for Logisim here:
https://github.com/logisim-evolution/logisim-evolution

I have included all documentation for the processor, as well as a sample program
